# expotsi

`expotsi` is a readable React Native Expo quick-reference package.

Install it with npm, then open files directly in `node_modules/expotsi`:

```bash
npm i expotsi
```

This package is intentionally documentation-only:

- No runtime exports
- No build system
- No dependencies
- Markdown notes with concise examples

## Structure

- `react/` → React basics
- `react-native/` → Core React Native concepts
- `navigation/` → Navigation patterns
- `state-management/` → State approaches
- `data/` → Data handling and async concepts

## Usage

After install, browse:

- `node_modules/expotsi/react`
- `node_modules/expotsi/react-native`
- `node_modules/expotsi/navigation`
- `node_modules/expotsi/state-management`
- `node_modules/expotsi/data`

Read any `.md` file as a quick concept reference.
